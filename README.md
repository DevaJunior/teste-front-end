# Teste Front-End Econverse

Este projeto foi desenvolvido como parte do processo seletivo para a vaga de Desenvolvedor Front-End na Econverse. O objetivo foi criar uma Landing Page (Vitrine de Produtos) fiel ao design do Figma, responsiva e com consumo de dados dinâmicos.

## 🚀 Tecnologias Utilizadas

* **React** + **TypeScript**
* **Vite** (Build tool e ambiente de desenvolvimento)
* **Sass (SCSS)** (Pré-processador CSS)
* **HTML5 Semântico**

## 🎯 Diferenciais Implementados

* **Pixel Perfect:** Respeito rigoroso ao layout, margens, cores e tipografia exigidos no Figma.
* **Responsividade:** Layout 100% adaptado para Desktop, Tablets e Smartphones (Mobile First em componentes-chave).
* **Zero Bibliotecas de UI:** Todo o CSS foi construído do zero com Sass, sem uso de frameworks como Bootstrap ou Tailwind, demonstrando domínio de CSS puro.
* **Componentização:** Estrutura organizada e reaproveitável.
* **SEO e Acessibilidade:** Uso adequado de tags semânticas (`<header>`, `<footer>`, `<section>`, `<nav>`).
